# ✨ Features

* 🚓 Moderne tablet UI
* 👮 Politiets MDT
* 📄 Straffeattest
* 🔍 Personopslag
* 🚗 Køretøjsopslag
* 📍 Dispatch integration
* 🎨 Nem konfiguration
* ⚡ Optimeret kode

---

# 📥 Installation

1. Download ressourcen.
2. Pak mappen ud.
3. Placer den i:

```text
resources/[VM]/VM_Politi-Tablet -- Valgfri sti
```

4. Tilføj til din `server.cfg`

```cfg
ensure VM_Politi-Tablet
ensure vrp
ensure oxmysql
```

5. Genstart serveren.

# ⚙️ Konfiguration

Alle indstillinger findes i ressourcens filer.

## config.lua

Her kan du ændre blandt andet:

* Framework
* Commands
* Keybinds
* Tilladelser
* Jobs
* Discord Webhooks
* Cooldowns
* Diverse indstillinger

---

## data.js

Her kan du ændre alt data der vises i tabletten.

Eksempel:

* Personer
* Betjente
* Rang
* Køretøjer
* Nummerplader
* Efterlysninger
* Rapporter
* Statistikker

Rediger blot værdierne og gem filen.

---

## style.css

Her kan hele designet ændres.

Du kan eksempelvis ændre:

* Farver
* Baggrund
* Knapper
* Animationer
* Skrifttyper
* Størrelser
* Ikoner

## script.js

Her ligger funktionerne til brugerfladen.

Du kan eksempelvis ændre:

* Knap funktioner
* Animationer

* Effekter

## html
